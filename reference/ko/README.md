## 핵심 논지

훈련 고리는 교리상 **설계로 닫혀 있다**:

> 계획 → 실행/훈련 → **AAR(교훈 포착)** → 평가 → **통합·공유** → 재훈련 → 다음 계획 주기

그러나 *교훈 포착*과 *교리/플레이북 실제 갱신* 사이에서 **실무상 끊긴다**:
- 공식 사후검토 보고서는 "작성되는 데 수년이 걸리거나, 아예 작성되지 않는다"(IAFC).
- 실행 항목은 개별 지휘자의 후속조치에 의존하며 종결까지 추적되지 않는다(Fire Engineering).
- 낮은 심리적 안전감이 원천에서 솔직한 포착을 억누른다(Fire Engineering).
- 교리는 결과를 통합할 선임 교관을 지정하고(FM 7-0 K-13) 교훈이 "항상 기록된다"고 하지만(K-42), 조직 전반의 플레이북 개정은 기계적으로 강제되지 않는다.

**NOEMA 웨지:** 솔직한 포착의 비용을 낮춘 뒤, **각 교훈/실행 항목을 종결까지 추적하고 표준 교리/플레이북에 반영**하는 자동 폐루프 계층 — 정확히 FM 7-0 K-13과 K-42–44 위에 자리한다.
