## 2. 공통점 (네 자료가 수렴하는 지점)

| 공통 요소 | 근거 |
|---|---|
| **"4질문" 학습 구조** (무엇이 일어났어야 / 실제로 / 왜 / 다음엔 어떻게) | FM 7-0 K-6 ≡ Fire Eng. 4질문 (1:1 대응); 핫워시·8단계는 암묵적 |
| **개인 비난 회피 → 상황 중심** | FM 7-0 K-38 "소대장, not 2LT Wilson" ≡ Fire Eng. "fault vs responsibility" |
| **교훈은 반드시 기록·공유** | FM 7-0 K-42·K-44, 핫워시 "공유할 권고", Fire Eng. "lessons distributed" |
| **군 특수작전 문화 기원** | Fire Eng. 명시; AAR 자체가 군 기원 |
| **즉시성 선호** (사건이 식기 전) | AAR "진행 중/직후", 핫워시 "가능한 한 빨리" |
| **공통 결핍: 종결까지 추적(track-to-closure) 부재** | 넷 다 "기록·공유"까지만 강제, 교리/플레이북 실제 갱신은 사람 손에 의존 |

## 3. 차이점 (서로를 보완하는 지점)

| 비교 축 | 한쪽 끝 | 반대쪽 끝 |
|---|---|---|
| **속도 vs 완결성** | 핫워시: 빠르지만 얕음 (수일) | 사후검토 보고서: 완결적이나 "수년 걸리거나 작성 안 됨" |
| **강제력 vs 자발성** | FM 7-0·8단계: 교리적 강제 | 핫워시: 서장 재량 제출 |
| **운영 표준 vs 인간 요인** | FM 7-0: *절차·역할* 표준화에 집중 | Fire Eng.: *심리적 안전감·두려움 장벽* 에 집중 |
| **단일 행사 vs 순환** | FM 7-0·Fire Eng.: 1회 AAR 운영 | 8단계: AAR를 주기에 끼워 *재훈련*으로 환류 |
| **개인 학습 vs 조직 전파** | FM 7-0(소부대 자기발견) | 핫워시(타 서장에게 게시·전파) |
| **심리적 안전감 깊이** | Fire Eng.: 가장 깊음 (계급 평탄화·세대별 두려움) | 핫워시·8단계: 거의 다루지 않음 |

## 4. 폐루프 단계별 커버리지 (가장 중요한 한 장)

각 자료가 **포착 → 추적 → 반영** 중 어디까지 책임지는가:

| 폐루프 단계 | FM 7-0 K | 핫워시 | 8단계 | Fire Eng. |
|---|:--:|:--:|:--:|:--:|
| ① 솔직한 **포착** | ◐ (절차) | ● (양식) | ○ | ● (심리안전) |
| ② 교훈 **기록·공유** | ● | ● | ◐ | ● |
| ③ 종결까지 **추적** | ○ | ○ | ○ | ○ |
| ④ 교리/플레이북 **반영** | ◐ (K-13 선임교관 지정뿐) | ○ | ◐ (재훈련까지만) | ○ |

`●` 명시적 책임 · `◐` 부분/역할만 지정 · `○` 공백

**한눈 결론:** 네 자료는 ①②(포착·기록)는 잘 덮지만, **③ 종결 추적은 어느 것도 책임지지 않고, ④ 조직 전반 반영은 "사람을 지정"하는 선에서 멈춘다.** 이 ③·④의 빈칸이 정확히 NOEMA 폐루프 웨지가 차지할 자리다.
